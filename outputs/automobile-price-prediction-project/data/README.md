# Data

This project uses the UCI Automobile dataset.

- Source page: https://archive.ics.uci.edu/dataset/10/automobile
- Raw data URL: https://archive.ics.uci.edu/ml/machine-learning-databases/autos/imports-85.data
- Local raw file: `data/raw/imports-85.data`

The raw file does not include a header row, so column names are assigned in `src/automobile_price/data.py`.
